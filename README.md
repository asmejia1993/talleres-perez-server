
## Description

Backend server para Talleres Perez & Perez

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# e2e tests
$ npm run test:e2e

```

## Docker container

* git clone https://github.com/asmejia1993/talleres-perez-server.git
* cd talleres-perez-server
* docker-compose build
* docker-compose up
